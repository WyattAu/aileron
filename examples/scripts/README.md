# Aileron Content Scripts

Place `.lua` files here or in `~/.config/aileron/scripts/` to inject custom JavaScript on matching web pages.

## Quick Start

1. Copy a script to `~/.config/aileron/scripts/`
2. Restart Aileron (or reload if hot-reload is implemented)
3. Visit a matching URL — the script runs automatically

## Examples

- `dark-mode.lua` — Force dark background on all pages
- `wide-github.lua` — Make GitHub repositories full-width
- `redirector.lua` — Auto-redirect Twitter/X to Nitter
